## Clone之後怎麼讓他run起來
   * 1. 到專案下，在command line 打 node -v 和 jest -v 確認是否已有安排，如果有出現版本，則是已安裝
   * 2. 在command line 打 npm i => 安裝完後會出現node_modules這個folder
   * 3. 在command line 打 npm run test 就會出現最後執行結果
